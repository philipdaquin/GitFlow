GitFlow CLI

A modern CLI tool for streamlining Git workflows. Built with Swift. Write less, commit more.

$ swift install https://github.com/philipdaquin/GitFlow.git
gitflow — zsh

Power Commands

Everything you need to dominate your git workflow. No more typing the same commands over and over.

⚡

Quick Commit

Stage all and commit with auto-generated or conventional messages. Just describe what you did.

gitflow commit --auto
🌿

Branch Management

List, create, delete, and rename branches with a single command.

gitflow branch create feature-x
📋

Changelog Generator

Generate changelogs from git tags. Supports conventional commits format.

gitflow changelog --conventional
↩

Undo Operations

Soft or hard reset with count support. Made a mistake? No problem.

gitflow undo --count 3
📦

Stash Helpers

List, push, pop, and drop stashes. Quick stash with messages.

gitflow stash push "WIP: feature"
🔄

Sync Commands

Fetch, pull, and push with a single command. Keep your repo in sync.

gitflow sync --pull

Command Reference

All available commands at a glance. Because who remembers everything?

commit

gitflow commit -m "msg" Commit with message
gitflow commit --auto Auto-generated message
gitflow commit --conventional Conventional commit

branch

gitflow branch list List all branches
gitflow branch create <name> Create new branch
gitflow branch delete <name> Delete branch
gitflow branch rename <name> Rename branch

changelog

gitflow changelog Generate changelog
gitflow changelog --from v1.0.0 From specific tag
gitflow changelog --conventional Conventional format

undo

gitflow undo Undo last commit
gitflow undo --count 3 Undo last N commits
gitflow undo --hard Hard reset

stash

gitflow stash list List stashes
gitflow stash push "msg" Push stash
gitflow stash pop Pop latest stash
gitflow stash drop Drop stash

misc

gitflow sync Fetch all
gitflow sync --pull Fetch and pull
gitflow info Repo info
gitflow config Configuration

Get Started

Install GitFlow in seconds. You just need Swift.

1

Clone Repository

Get the source code

git clone https://github.com/
philipdaquin/GitFlow.git
2

Build & Install

Compile with Swift

swift build -c release &&
cp .build/release/GitFlow /usr/local/bin/gitflow
3

Or Use Swift Install

The easy way

swift install https://github.com/
philipdaquin/GitFlow.git

⚠️ Requirements

You need Swift installed on your system. macOS comes with Swift pre-installed. On Linux, install via swift.org
